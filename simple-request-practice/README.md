# Simple API Request Practice

This is a very small practice for writing `fetch()` requests.

Students type requests in `practice.js`, run the file, and see the output in the
terminal.

This uses JSONPlaceholder, a free fake API:

```txt
https://jsonplaceholder.typicode.com
```

## Start

```bash
cd simple-request-practice
node practice.js
```

## What To Practice

- `GET` reads data
- `POST` creates data
- `PUT` replaces data
- `PATCH` updates part of data
- `DELETE` removes data

## Files

```txt
practice.js   students type requests here
solution.js   answer key
```
